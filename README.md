# Пульт дня

Личный сайт-пульт на рабочий день: расписание 10:00–18:00 с подсветкой текущего блока, чек-лист, Pomodoro-таймер (в работе).

Живёт на GitHub Pages: https://konicaru.github.io/day-runner/

## Как запустить локально

Открыть `index.html` двойным кликом в браузере. Никакой сборки, сервера и npm не нужно.

## Как обновить расписание

Правится руками в `schedule.js`. Это обычный JS-массив: каждый блок — объект с `start`, `end` (формат `"HH:MM"`), `title` и массивом `items`. У каждого `item` должен быть стабильный `id` (строка) и `text`. По `id` хранятся галочки в `localStorage`.

После правки — обновить страницу в браузере. Для публикации — `git commit` + `git push`, Pages подхватит сам.

## Структура

- `index.html` — разметка
- `styles.css` — стили
- `schedule.js` — данные расписания
- `app.js` — логика (часы, рендер, подсветка)
